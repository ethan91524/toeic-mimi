設定 Google 登入
需要先設定 Firebase，才能使用 Google 帳號儲存進度
1
前往 Firebase Console，用 Google 帳號登入，建立新專案
2
在專案中點 「Authentication」 → 「Sign-in method」→ 啟用 「Google」
3
在專案中點 「Firestore Database」 → 建立資料庫（選擇 production mode），然後在「規則」頁貼上：
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }
4
到 「專案設定」（齒輪圖示）→「一般」→「你的應用程式」→ 新增 Web 應用程式，複製 firebaseConfig 的內容
5
用文字編輯器（如記事本）打開這個 HTML 檔，找到 FIREBASE_CONFIG 區塊，把 YOUR_* 的值替換成你剛剛複製的內容，存檔即可
